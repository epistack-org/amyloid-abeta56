# amyloid-abeta56

**Alzheimer's amyloid — the Aβ*56 corroboration cascade**

Self-contained, verify-only mirror of the `amyloid-abeta56` case bundle (CASE-REPO-SPEC.md). It ships 10 pre-minted `records/` in the pinned assembly order (`CASE.json` `records`), a `PROVENANCE.md` vetting record, an `engine.pin` and 4 version-of-record source excerpt(s) under `sources/`, and an `expected.digest`. Verify standalone with `cairn cases verify .` and recompute the digest against `expected.digest`; no `build.py`, no signing seed, no minting required.
